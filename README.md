# Program Luas Persegi Panjang (Java)

Program ini dibuat menggunakan bahasa Java untuk menghitung luas persegi panjang menggunakan input visual (Dialog Form).

## Fitur Program
- Input nilai panjang menggunakan dialog
- Input nilai lebar menggunakan dialog
- Menampilkan hasil luas persegi panjang

## Rumus
luas = panjang × lebar

## Contoh
panjang = 15  
lebar = 30  

luas = 450
